This project is based on Reactjs and mongodb as a database. The moto of creating this project is to share the memories amongst the users of app which are able to connect with each other and can communicate via the comment section of posts. The Posts shared by a particular user can be viewed amongst all users who visits the site but only the users who are signed in will be able to comment and like the that post. Also the search functionality helps the user to find the post of a particular user.
